# Legacy scripts

These scripts are **superseded** by the packaged `src/obfviewer/` module and are
kept here for reference only.

| File | Description | Replaced by |
|------|-------------|-------------|
| `obf_viewer_v1.py` | Original single-file Tkinter 2D viewer | `obfviewer.gui` |
| `obf_viewer_3d.py` | Standalone 3D PyVista viewer | `obfviewer.combined` |
| `obf_viewer_3d_fast.py` | Ultra-fast standalone 3D viewer (prototype) | `obfviewer.combined` |

Do **not** use these for new work.
