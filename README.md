# obfviewer
Extension of Freemelt's obpviewer to enable viewing of complete obf-format build jobs.
This script is based on the version with some improvements over the original already, taken from [wiberganton's OBPlanner](https://github.com/wiberganton/OBPlanner).

The script unpacks the obf file into a temporary folder. It then creates data structures containing information on the layer stack so that it can be shown in proper sequence.
